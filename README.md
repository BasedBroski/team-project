# Keyboard Shortcuts Finder

## Team Members
- Christian Miller                   
- Garett Clark

## Description
A web app that allows users to search and filter keyboard shortcuts.

## Features
- Search shortcuts
- Filter by category
- Data loaded using JavaScript Fetch API

## How to Use
1. Click the live app link.
2. Select a category or search shortcut by name
3. Get results
